## Naming Convention

1️⃣ **Folder** naming convention

Use **kebab-case** (lowercase + hyphen)

```
design-patterns/
class-relationships/
```

2️⃣ **Markdown file** naming convention

Use **kebab-case** (lowercase + hyphen)

```
single-responsibility-principle.md
```

3️⃣ **Code file** naming convention

Use Language standard naming convention

Let's say for **Java**, then 

use **PascalCase** for **class name**, **camelCase** for **method name** and **UPPER_SNAKE_CASE** for **constants**.
